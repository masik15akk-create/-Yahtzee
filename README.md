🎲 Yahtzee – Покер на костях 
Классическая игра в кости «Яхтзи» с полным набором правил, подсчётом очков, игрой против компьютера и сохранением рекордов!
Бросайте 5 кубиков, комбинируйте комбинации и набирайте максимальные очки.
Реализована на 7 языках программирования с единым интерфейсом и расширенными возможностями.

🚀 Возможности
5 кубиков – от 1 до 6 очков, до 3 бросков за ход.

13 категорий для подсчёта очков (единицы, двойки, тройки, четвёрки, пятёрки, шестёрки, три одинаковых, четыре одинаковых, фулл-хаус, малый стрит, большой стрит, шанс, яхтзи).

Игровые режимы:

vs – два игрока (по очереди).

ai – игра против компьютера (простой ИИ выбирает лучшую категорию).

Подсказка – компьютер показывает лучшую категорию для текущего набора костей.

Статистика – количество игр, побед, поражений, лучший счёт.

Сохранение – рекорды и статистика сохраняются в локальный файл.

Цветной вывод – кости, категории и результаты выделены для удобства.

Интерактивность – выбор костей для повторного броска, просмотр доступных категорий.

📖 Использование
Синтаксис (единый для всех версий):

bash
<команда> <режим> [опции]
Режимы
vs – два игрока.

ai – игра против компьютера.

Опции
Опция	Описание
-p, --players <N>	Количество игроков (для vs, по умолчанию 2)
-n, --names <name1,name2,...>	Имена игроков через запятую
-s, --stats	Показать статистику и выйти
-r, --reset	Сбросить статистику
-h, --help	Справка
Примеры
bash
# Игра против компьютера
python yahtzee.py ai

# Два игрока
python yahtzee.py vs

# Игра с именами
python yahtzee.py vs -n "Алексей,Мария"

# Показать статистику
python yahtzee.py -s
Управление
После броска введите номера кубиков, которые хотите оставить (через пробел), или 0 чтобы оставить все.

Введите roll для повторного броска (до 3 раз).

Выберите категорию из списка, введя её номер.

q – выход.

🧠 Правила игры
Категории и подсчёт очков
Категория	Описание	Очки
1–6 (верхняя секция)	Сумма костей с соответствующим значением	Сумма значений
Три одинаковых	Не менее 3 костей одного номинала	Сумма всех костей
Четыре одинаковых	Не менее 4 костей одного номинала	Сумма всех костей
Фулл-хаус	3 кости одного номинала и 2 другого	25
Малый стрит	4 последовательных значения (1-4, 2-5, 3-6)	30
Большой стрит	5 последовательных значений (1-5 или 2-6)	40
Шанс	Любая комбинация	Сумма всех костей
Яхтзи	Все 5 костей одинаковые	50
Ход игры
Игрок бросает 5 кубиков.

Выбирает, какие кубики оставить (0 – оставить все, или номера кубиков через пробел).

Делает до 3 бросков, каждый раз решая, какие кости оставить.

Выбирает категорию для записи результата.

Если категория уже использована, игрок получает 0 очков за неё (но ход не засчитывается).

Игра заканчивается, когда все категории заполнены.

Побеждает игрок с наибольшим суммарным счётом.

✨ Дополнительные фичи
Подсказка ИИ – компьютер анализирует все категории и рекомендует лучшую.

Статистика – сохраняется количество игр, побед, поражений и лучший счёт.

Цветной вывод – кубики отображаются в цвете (1-6 разными цветами).

Автосохранение – рекорды и статистика хранятся в ~/.yahtzee_stats.json.
